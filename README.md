## 🚀 Deployment Options

### Option 1: Deploy to Vercel (Easiest & Fastest) ⭐

**What is Vercel?** Vercel is a free platform to host your website. Your portfolio will be live in minutes!

**Steps:**

1. **Create a GitHub Account** (if you don't have one)
   - Go to github.com
   - Click "Sign up"
   - Fill in username, email, password
   - Verify email

2. **Create a GitHub Repository**
   - Log in to github.com
   - Click "+" icon → New repository
   - Repository name: `portfolio` (or any name)
   - Description: "AI/ML Engineer Portfolio"
   - Click "Create repository"

3. **Upload Files to GitHub**
   - In your repository, click "Add file" → "Upload files"
   - Drag and drop these files:
     - portfolio.html
     - package.json
     - server.js
     - vercel.json
     - .gitignore
   - Click "Commit changes"

4. **Deploy to Vercel**
   - Go to vercel.com
   - Click "Sign up" → "Continue with GitHub"
   - Authorize Vercel to access GitHub
   - Click "New Project"
   - Select your `portfolio` repository
   - Click "Import"
   - Click "Deploy"
   - Wait 1-2 minutes ✅ Your portfolio is live!

5. **Get Your Portfolio URL**
   - After deployment, Vercel shows your live URL
   - Example: `https://portfolio-your-username.vercel.app`
   - Copy this link and share it!

### Option 2: Deploy to Netlify

**Steps:**

1. **Upload to GitHub** (same as Option 1, steps 1-3)

2. **Deploy to Netlify**
   - Go to netlify.com
   - Click "Sign up" → "GitHub"
   - Authorize Netlify
   - Click "New site from Git"
   - Select your GitHub repository
   - Click "Deploy site"
   - Wait for deployment ✅

3. **Your Netlify URL will appear**

---

### Option 3: Deploy to GitHub Pages (Free, Simplest)

**Steps:**

1. **Create Repository** (same as Option 1, steps 1-2)

2. **Upload Files to GitHub** (same as Option 1, step 3)

3. **Enable GitHub Pages**
   - In your repository, go to Settings
   - Scroll to "Pages" (left sidebar)
   - Source: Select "main" branch
   - Click "Save"
   - Wait 1-2 minutes ✅

4. **Your Portfolio URL**
   - Example: `https://your-username.github.io/portfolio`

---

### Option 4: Run Locally (For Testing)

**Requirements:**
- Node.js installed (download from nodejs.org)

**Steps:**

1. **Open Terminal/Command Prompt**

2. **Navigate to Portfolio Folder**
   ```
   cd path/to/portfolio
   ```

3. **Install Dependencies** (optional)
   ```
   npm install
   ```

4. **Start Server**
   ```
   npm start
   ```

5. **Open Browser**
   - Go to http://localhost:3000
   - Your portfolio opens! 🎉

## 🔧 Troubleshooting

**Portfolio not loading?**
- Wait 2-3 minutes for deployment to complete
- Clear browser cache (Ctrl+Shift+Del)
- Try different browser

**Links not working?**
- Check URL spelling in portfolio.html
- Make sure links start with https://

**Want custom domain?**
- Vercel/Netlify: Buy domain from namecheap.com
- Connect to Vercel/Netlify in settings
- Costs ~$10/year
